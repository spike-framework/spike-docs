Automatic selectors

Spike Framework have selectors feature which can handle multiple the same id and generate automatic jQuery selectors for views.

Selectors are created for current context of controller, component or modal module.
Then you can declare the same id attributes in different templates of those modules without taking care of any conflicts.

Example retrieving jQuery selector:

                'import $this as app.controller.Home';

                app.controller.register("Home", {

                    init: function(params) {

                        $this.selector.login() //returns jQuery selector for id="login"

                    }

                });

Spike will create dynamic id attributes values for example id="login" will looks like id="login-f5df4g-5g6g6-g456h"

Binding events, setting values

Developer have to understand, that selector object contains set of functions which ones returns jQuery selector

Also developer don't have to care about efficient selecting, because selecting via id attribute is the fastest way in general, and Spike also cache selectors per every controller lifecycle.


Example binding jQuery click event:

                'import $this as app.controller.Home';

                app.controller.register("Home", {

                    init: function(params) {

                        $this.selector.login().click(function(){
                            app.log('button clicked');
                        });

                    }

                });

All functions provided by jQuery are available.

Example setting HTML or value usage:
                'import $this as app.controller.Home';

                app.controller.register("Home", {

                    init: function(params) {

                        $this.selector.login().html('Button name');
                        $this.selector.emailInput().val('myemail@test.com');

                    }

                });

Names selector

Spike also allows to select by names but names are not changing

Example names selectors usage:

                'import $this as app.controller.Home';

                app.controller.register("Home", {

                    init: function(params) {

                       $this.selector.names.username().val('user123');
                       $this.selector.names.password().val('1111');

                    }

                });
