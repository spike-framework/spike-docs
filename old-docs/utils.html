Util module

Spike Framework divide application into separately modules and one of them is util.

util should be used as storage for not business functions and another parts of code which ones not fit into any another structure

Utils are singletons

Utils preferable directory structure should be as follows:

  • src/app (value from app.config.rootPath)
    • util
      • image.util.js
      • list.util.js

Also naming convention should be keeped as:

Utils file name should be named UTILS_NAME.util.js

Example declaration of util:

                'import $this as app.util.List';

                app.util.register("List", {

                    limit: function(list, limit){
                        return list.slice(0, limit);
                    }

                });

Then util can be used with static reference as well in imports in any another modules: 

               'import $listUtil as app.util.List'; //import way

                app.util.Status //static way
