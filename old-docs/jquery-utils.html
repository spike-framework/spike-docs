jQuery build-in utils

Spike has build-in jQuery utils which can be very helpful for daily development and are also used by Spike itself.

jQuery extensions

set
Abstracted version of set of typical jQuery functions like html, val, prop and attr
Example usage: 

                        ...

                        $this.selector.formTitle().set('Subscribe');
                        $this.selector.emailLabel().set('E-mail: ');
                        $this.selector.emailInput().set('heniek@okk.com');
                        $this.selector.acceptRegulation().set(true);
                        $this.selector.regulationsLink().set('/regulations');
                        $this.selector.avatarImage().set('/default_avatar.png');

                        ...

Example view:

                            See regulation

Also spike-val attribute can be used to setting objects:
Example usage: 

                        ...

                        $this.selector.data().set({
                            id: 10,
                            name: 'Michael',
                            surname: 'Jackson'
                        });

                        ...

serializeObject

Function serializes form fields into single JavaScript object based on fields names.

Example usage: 

                        ...

                        var formData = $this.selector.loginForm().serializeObject()

                        //formData == { email: 'ok@test.com', password: '1111' }

                        ...

attrs

Function serializes DOM element attributes into map

Example usage: 

                        ...

                        var elementAttrs = $this.selector.person().attrs()

                        //elementAttrs == { personId: 2, personName: 'David' }

                        ...
