Getting started

Spike Framework was created to handle JavaScript Web & Mobile projects in simple way

Spike Framework is written simple, clear code without any magic and tens of dependencies as in another frameworks.

Read more about history and current status on about page

First steps

  • Clone ready to use starter template with basic features
    git clone https://github.com/dsenko/spike-framework-starter.git
  • Prepare project installing dependencies
    npm install && bower install
  • Build application and serve locally with watching changes on localhost:2111
    grunt dev
  • Write your application to be used as web or mobile or both and make production version
    grunt build

Read project structure and start building your own application in simple way!
