Spike CLI

Spike Compiler offers own simple CLI to creating any module like component, controller, partial, util, service, modal.

Spike CLI can be executed with Java as any Spike Compiler methods:

java -jar spike-compiler.jar cli DIST_PATH MODULE_TYPE MODULE_NAME

Allowed commands examples:

Controller

Example:

java -jar spike-compiler.jar cli src/app/controller/ controller Home

Component

Example:

java -jar spike-compiler.jar cli src/app/component/ component Menu

Partial

Example:

java -jar spike-compiler.jar cli src/app/partial/ partial PostList

Modal

Example:

java -jar spike-compiler.jar cli src/app/modal/ modal LoginModal

Service

Example:

java -jar spike-compiler.jar cli src/app/service/ service Posts

Util

Example:

java -jar spike-compiler.jar cli src/app/util/ util Utils
